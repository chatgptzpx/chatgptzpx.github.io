M5StickPlus2

M5StickPlus2 - это компактное IoT устройство с микроконтроллером ESP32-S3, цветным дисплеем и множеством встроенных сенсоров. Идеально подходит для разработки прототипов и DIY проектов.

Технические характеристики

Параметр Значение
Процессор ESP32-PICO-V3-02 (240 МГц dual core)
PSRAM 2 МБ
Flash память 8 МБ
Дисплей 1.14" IPS LCD (240×135), ST7789v2
Сеть Wi-Fi 802.11 b/g/n, Bluetooth
Датчики IMU (MPU6886), MIC (SPM1423), RTC (BM8563)
Кнопки 3 программируемые кнопки
Светодиоды Зеленый (индикатор сна), красный (G19, общий с ИК-передатчиком)
Питание Аккумулятор 200 мАч @ 3.7В
Вход питания 5В @ 500мА
Интерфейсы TypeC x 1, GROVE (I2C+I/0+UART) x 1
GPIO G0, G25/G36, G26, G32, G33
Дополнительно Встроенный зуммер, 2.4G 3D антенна
Рабочая температура 0°C до 40°C
Материал корпуса Пластик (PC)
Размеры 48 x 25 x 13 мм
Вес 17 г

Особенности

  • Микроконтроллер ESP32-PICO-V3-02
  • 1.14-дюймовый TFT-дисплей
  • Встроенный микрофон SPM1423
  • IMU датчик MPU6886
  • Инфракрасный передатчик
  • Интерфейс GROVE (I2C+I/0+UART)
  • Встроенный зуммер
  • Аккумулятор 200 мАч
  • Поддерживаемые платформы разработки:
    • Arduino
    • UIFlow
    • ESP-IDF
    • MicroPython

T-Embed

T-Embed - это встраиваемое устройство с круглым дисплеем и контроллером ESP32-S3, оснащенное поворотным энкодером и кольцевой RGB подсветкой. Идеально подходит для проектов с круговым интерфейсом и аудио функциями.

Процессор ESP32-S3
Сеть Wi-Fi IEEE 802.11 b/g/n, Bluetooth 5, Bluetooth mesh
Дисплей 1.9" IPS LCD (170×320), ST7789, яркость 350 кд/м²
Управление Поворотный энкодер с кнопкой подтверждения (24 фиксации, 12 импульсов/360°)
Аудио Микрофон x 2, динамик 8 Ом @ 1Вт
Подсветка Кольцевая RGB подсветка x 7
Интерфейсы GROVE x 1, GPIO 2.54 x 8 пин, слот для TF карты, Type-C USB
Материал ABS
Дополнительно Схема защиты заряда/разряда аккумулятора, батарейный интерфейс 1.25 мм

Особенности

  • Микроконтроллер ESP32-S3
  • Wi-Fi IEEE 802.11 b/g/n; Bluetooth 5, Bluetooth mesh
  • 1.9-дюймовый IPS цветной TFT LCD дисплей
  • Поворотный энкодер со встроенной кнопкой подтверждения
  • 2.54 x 8pin GPIO интерфейс
  • Кольцевая RGB подсветка из 7 светодиодов
  • Два микрофона для записи звука

Flash память 8 МБ
Хранилище SD карта
Клавиатура 56 клавиш (4 x 14)
Кнопки 1 кнопка сброса + 1 пользовательская кнопка
Дисплей ST7789V2 @ 1.14", 240 x 135 пикселей
Аудио Динамик 8Ω @ 1Вт (I2S @ NS4168), MEMS микрофон (SPM1423)
Аккумулятор 120 мАч + 1400 мАч (в основании)
Инфракрасный порт На 0°: 410 см, на 90°: 66 см, на 45°: 170 см
Ток в режиме сна DC-4.2В @ 0.15мкА
Рабочий ток Режим ИК-передачи: DC-4.2В/148.07мА
Режим клавиш: DC-4.2В/138.93мА
Интерфейсы Grove x 1
Рабочая температура 0-40°C
Размеры 84.0 x 54.0 x 19.7 мм
Вес 90.0 г

Особенности

  • Микроконтроллер StampS3A (ESP32-S3)
  • Клавиатура с 56 клавишами (4 x 14)
  • 1.14-дюймовый TFT экран
  • Динамик и цифровой MEMS микрофон SPM1423
  • Инфракрасный излучатель для управления
  • Интерфейс HY2.0-4P (Grove)
  • Слот для Micro SD карты
  • Встроенный аккумулятор 120 мАч и 1400 мАч в основании
  • Основание с магнитами
  • Совместимость с креплениями LEGO

Flipper Zero
