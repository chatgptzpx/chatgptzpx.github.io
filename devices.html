ESP32 Marauder v7

ESP32 Marauder v7 - это официальное устройство среднего размера с 5-позиционным тактильным переключателем. Оснащено встроенным GPS, управлением LiPo аккумулятором, индикатором заряда и внешними антеннами для 2.4GHz и GPS. Устройство имеет 2.4" ILI9341 экран и 5-позиционный тактильный переключатель для навигации по функциям с помощью джойстика. Marauder v7 полностью управляется через Serial с помощью интерфейса командной строки Marauder.

Питание устройства

Marauder v7 может получать питание из двух источников:

  • USB-питание: Должно быть 5В
  • LiPo аккумулятор: 3.7В 1S с 2-контактным JST-разъемом 2мм

Подключите LiPo аккумулятор к встроенному разъему JST. Убедитесь, что полярность аккумулятора соответствует маркировке на плате Marauder v7. При подключенном USB-питании и/или аккумуляторе переключите выключатель питания на левой стороне корпуса в положение ВКЛ. После этого вы должны увидеть загрузочный экран Marauder.

Внешние антенны

2.4GHz антенна: Входящую в комплект антенну 2.4GHz можно снять, чтобы открыть разъем SMA female. Этот разъем примет любую антенну 2.4GHz с разъемом SMA male. На корпусе под разъемом антенны есть маркировка для правильного размещения антенны.

GPS антенна: Входящую в комплект GPS-антенну можно снять, чтобы открыть разъем SMA female. Этот разъем примет любую GPS/GNSS антенну с разъемом SMA male. На корпусе под разъемом антенны есть маркировка для правильного размещения антенны.

USB Killer V4

USB Killer - это устройство, предназначенное для стресс-тестирования аппаратного обеспечения. Принцип работы основан на заборе энергии из USB-порта, её умножении и последующем разряде в линии данных, что обычно приводит к выведению из строя незащищенных устройств.

Принцип работы

Устройство при подключении накапливает электрический заряд от USB-порта, а затем мгновенно разряжает его обратно в порт, создавая высоковольтный импульс. Этот процесс повторяется несколько раз в секунду, вызывая повреждение электронных компонентов, подключенных к USB-контроллеру устройства.

Версии USB Killer V4

  • USB Killer V4 Pro - полный набор функций, включая дистанционное управление
  • USB Killer V4 Basic - базовая версия с магнитным триггером
  • USB Killer V4 Classic - классическая версия с режимом "plug & zap"

Режимы атаки

Режимы доставки

Одиночный импульс: USB Killer V4 отправляет один импульс (5 разрядов)

Непрерывный импульс: USB Killer V4 отправляет непрерывные импульсы до остановки

Способы активации

Дистанционная активация: Запуск атаки через пульт дистанционного управления на расстоянии до 100м от устройства

Активация через смартфон: Управление и запуск атак через приложение для Android и iOS

Атака по таймеру: Планирование даты и времени активации USB Killer. Устройство может оставаться в спящем режиме без питания более 200 дней

Магнитная активация: Активация USB Killer с помощью магнита (стильное и скрытное магнитное кольцо в комплекте)

Классический режим: Активация мгновенно при подключении к USB-порту

Применение

USB Killer используется для проверки и тестирования защитных систем от перенапряжения в USB-портах. Профессионалы в области кибербезопасности, пентестеры и правоохранительные органы используют это устройство для:

  • Тестирования систем защиты USB-портов
  • Проверки отказоустойчивости критически важной инфраструктуры
  • Оценки безопасности медицинского оборудования
  • Проверки систем контроля доступа на устойчивость к атакам
  • Демонстрации уязвимостей в рамках обучения по кибербезопасности
